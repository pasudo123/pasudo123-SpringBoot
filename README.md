# SpringBoot + Vue.js
* pasudo123`s SpringBoot Study Sample Code
* Restful Application + Vue Application
* 뭐라도 만들며 공부하자

### Vue 설치
* vue cli 3.x
* 순서대로 진행하였음
  1. vue 프로젝트 생성
  2. axios 플러그인 설치
  3. vuex 설치 (의존성 주입 : package.json)
  4. vue-router 설치 (의존성 주입 - pakcage.json)
  5. vuetify 플러그인 설치
  
```code
vue create frontend

vue add axios

npm install vuex --save

npm install vue-router

vue add vuetify
```
